# SaaS Pipeline Analysis (Python)

## Overview
This project analyses SaaS sales pipeline data to identify factors affecting conversion rates.

## Features
- Data loading using Pandas
- Conversion rate calculation
- Analysis by Technology and Sales Channel
- Visualization using Matplotlib
- Automated chart generation

## Project Structure
data/ → dataset (Excel)
src/ → analysis logic
output/ → generated charts
main.py → entry point

## How to Run

1. Install dependencies:
pip install pandas matplotlib openpyxl

2. Run:
python main.py

## Example Use Case
- Analyze sales conversion performance
- Identify high-performing segments
- Support business decision-making
